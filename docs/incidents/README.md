## Guía para nuevos incidentes

Todo incidente se documenta antes de pasar a la siguiente tarea (Guardrail AI Agents #9).

Cada reporte debe incluir:
- Fecha de detección
- Severidad (Baja / Media / Alta / Crítica)
- Estado actual
- Resumen ejecutivo
- Hallazgos técnicos
- Decisión tomada
- Lecciones aprendidas
- Referencias cruzadas a otros incidentes o documentos relacionados
