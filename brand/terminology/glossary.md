# Terminología oficial eLIGHTS.cl

## Nombre de marca (REGLA CRÍTICA)

La marca es **`eLIGHTS.cl`** — con el `.cl` integrado como parte inseparable del nombre.

### Formato exacto

- `e` → minúscula
- `LIGHTS` → todo mayúsculas
- `.cl` → minúscula, sin espacios
- Resultado: `eLIGHTS.cl`

### Por qué incluye `.cl`

Decisión estratégica deliberada: afirma chilenidad, refuerza confianza local en un rubro donde los clientes valoran proveedores nacionales con respaldo SEC y RUT chileno, y diferencia de importadores grises o marcas internacionales sin localización.

### Datos complementarios

- **Razón social:** eLIGHTS SpA
- **URL del sitio:** https://elights.cl
- **Nueva plataforma:** https://nuevo.elights.cl

### Formas prohibidas

| Forma incorrecta | Problema |
|---|---|
| `eLIGHTS` | Omite el `.cl` — no es la marca |
| `Elights.cl` | Primera letra en mayúscula — incorrecto |
| `ELIGHTS.CL` | Todo mayúsculas — incorrecto |
| `elights.cl` | Todo minúsculas — incorrecto cuando se usa como marca (aceptable sólo como URL literal) |
| `E-lights.cl` | Lleva guión — incorrecto |
| `eLights.cl` | `LIGHTS` no en mayúsculas — incorrecto |
| `eLIGHTS .cl` | Lleva espacio — incorrecto |

### Advertencia para agentes de IA

El error más frecuente al generar contenido para esta marca es omitir el `.cl` (tratarlo como si fuera solo un dominio y no parte del nombre). Este error causó un incidente real documentado en `../logo/NO-USAR/`. Antes de generar material, verifica dos veces que el nombre aparece como `eLIGHTS.cl` en todos los lugares donde se menciona la marca.

## Promesas prohibidas

- "Siempre en stock" / "stock permanente" / "stock ilimitado"
- "10+ años de experiencia"
- "Despacho inmediato" (sin calificar zona)
- "Garantía de por vida"
- "El precio más bajo de Chile" (no validable)
- "100% certificado" (ambiguo)
- Certificaciones que no tenemos (UL, ETL, ENERGY STAR, etc.)

## Tono de voz

- **Tuteo siempre** ("te ayudamos", "tu proyecto")
